四、RadioButton单选按钮和按钮组

将多个RadioButton用RadioButtonGroup组括起来才能实现单选的效果，即：

Java代码：
`
RadioGroup rg = (RadioGroup)findViewById(R.id.xx)
rg.setOnCheckedChangeListener(new RadioGroup.OnCheckedChangeListener(){
public void OnCheckedChange(RadioGroup group,int checkedID){
RadioButton r=(RadioButton)findViewById(checkedID); //通过参数checkedID取出所选单选框
r.getText(); //获取所选单选框的文本内容
}
})
